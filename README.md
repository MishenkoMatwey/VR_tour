# Виртуальная экскурсия по кафедре  МОЭВМ
`Для запуска программы :`<br>
- git clone 
- cd в только что скачанную папку "cd VR_tour"
- npm install
- npm start 
- откройте браузер по адресу http://localhost:8081/index.html
